# ColorWizard Desktop

ColorWizard Desktop is a painting and color literacy tool designed for offline use. It helps users understand color relationships, build palettes, and practice color mixing in a focused, distraction-free environment.

## Purpose

ColorWizard exists to teach and support color literacy for painters, illustrators, and anyone working with physical media. It translates digital exploration into knowledge applicable to real-world materials.

## Core features

- Interactive color wheel with mixing simulation
- Palette creation and management
- Color harmony visualization
- Offline reference library
- Export to user-readable formats (JSON, plain text)

## What it is not

ColorWizard is not:

- A photo editor
- A vector graphics tool
- A design asset manager
- A cloud-synced palette service
- A collaborative platform

It is a focused tool for understanding and working with color.

## Technical approach

- Fully offline desktop application
- Local data storage only
- No accounts or authentication
- No network requests
- Stable, minimal dependencies
