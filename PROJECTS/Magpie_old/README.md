# Magpie

Magpie is an embroidery and pattern translation tool designed for offline use. It helps users convert images and drawings into stitch patterns, manage thread palettes, and prepare designs for physical embroidery work.

## Purpose

Magpie exists to bridge digital design and physical embroidery. It translates visual ideas into practical stitch instructions for hand embroidery and simple machine work.

## Core features

- Image to stitch pattern conversion
- Thread palette management with common floss libraries
- Stitch count estimation
- Pattern export in user-readable formats
- Offline reference for stitch types

## What it is not

Magpie is not:

- A photo editor or illustration tool
- A machine embroidery file converter for industrial formats
- A pattern marketplace or social platform
- A cloud storage service
- A substitute for embroidery knowledge

It is a focused tool for preparing and organizing embroidery work.

## Technical approach

- Fully offline desktop application
- Local data storage only
- No accounts or authentication
- No network requests
- Stable, minimal dependencies
