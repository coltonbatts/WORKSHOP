# Color Wizard - Marketing & Launch

**Project:** colorwizard.app  
**Status:** Built, not launched  
**Timeline:** Built in last month (Jan 2025)

## What It Is

A color analysis tool for artists and crafters:
- Upload any image, click any color
- Get DMC embroidery thread matches with exact codes
- Oil paint mixing recipes (spectral-based, real physics)
- Value/chroma analysis (Munsell system)
- Canvas grid overlay for planning

## Target Audiences

1. **Embroiderers** - DMC thread matching from reference photos
2. **Painters** - Oil paint mixing recipes with ratios
3. **General artists** - Color analysis, palette building

## Current State

- Fully functional
- Never marketed (only shown to Colton + Kait)
- Free access (no monetization yet)
