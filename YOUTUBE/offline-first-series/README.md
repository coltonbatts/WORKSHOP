# Offline-First Series

Building a YouTube channel around offline-first creative tools, local AI, and artist autonomy.

## The Mission

Show artists how to own their tools, run local models, and break free from subscription dependencies.

**Core Thesis:** Eventually everyone will run their own offline agents and local models — like a personal creative studio. I'm early to this space and want to be the one showing artists the way.

## Series Structure

Each episode follows a tight format:
1. **Hook (15s)** — Relatable friction, punchy
2. **Problem (30s)** — What's broken with the current model
3. **Fix (1min)** — Solution + demo
4. **Stack (1.5min)** — Technical walkthrough
5. **Vision (1min)** — Bigger picture, artist-focused thesis

**Target runtime:** 4-5 minutes per episode
