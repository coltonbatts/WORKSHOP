# Equipment & Setup

## Recording

### Screen Capture
- **OBS Studio** (free, solid, cross-platform)
- **CleanShot X** (macOS, cleaner captures, built-in annotation)

### Audio
- Your existing microphone — just test levels
- **Audacity** (free) for cleanup and leveling
- Tips:
  - Record in a quiet room
  - Speak clearly, slightly slower than natural pace
  - Keep mic 6-12 inches from mouth

### Format
- **Resolution:** 1080p or 4K
- **Framerate:** 60fps for smooth code scrolling
- **Codec:** H.264 or HEVC (macOS native)

## Editing

### Software Options
- **DaVinci Resolve** (free, powerful, steeper learning curve)
- **CapCut** (free, easier, great for quick cuts)
- **Final Cut Pro** (paid, macOS only, industry standard)

### Quick Edits
- Trim dead air
- Level audio
- Add simple text overlays for code snippets
- B-roll: loading icons, subscription screenshots

## Workflow

1. **Plan:** Review script/outline
2. **Record screen:** Demo portions, code walkthrough
3. **Record voiceover:** Easier to re-record flubs
4. **Edit:** Sync audio + video, clean up
5. **Export:** YouTube recommended settings (H.264, 1080p or 4K)

## YouTube Settings

- **Title format:** "I Built [X] Offline — Here's How"
- **Description:** Include tags, links to code, call to action
- **Thumbnail:** High contrast, text overlay, face (optional)
- **End screens:** Link to next episode, subscribe button
